# 🛠️ Practical Scripts

This directory contains practical Bash scripts focused on automation, system management, and cybersecurity fundamentals.

Each script demonstrates real-world use cases and reinforces Linux command-line skills.

---

## 📦 Available Scripts

### 📁 [Backup Script](./backup-script)
Creates timestamped backups of a directory.

- Automates file backup  
- Prevents overwrite with timestamps  
- Simple and effective  

---

### 📁 [Compress Script](./compress-script)
Compresses a directory into a `.zip` file.

- Fast file compression  
- Generates timestamped archives  
- Useful for storage and transfer  

---

### 📁 [System Monitor Script](./monitor-script)
Displays system resource usage in real time.

- CPU usage  
- Memory consumption  
- Disk space overview  

---

## 🚀 Purpose

These scripts are designed to:

- Practice Bash scripting  
- Automate common Linux tasks  
- Build a practical cybersecurity skill set  

---

## 📚 Key Learnings

- Bash scripting fundamentals  
- Process and system monitoring  
- File management and automation  
- Use of Linux utilities (`cp`, `zip`, `top`, `df`, `free`)
